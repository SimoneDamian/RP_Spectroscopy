When the user clicks on the "Save reference line" button in @add_reference_line_page.py, this has to trigger an add_reference_line method in the service manager (@service_manager.py) with all the informations the user inserted in the textboxes plus the board name that is fixed from the conneciton to the board. Give a look at the other reference lines methods in the @service_manager.py in order to understand how to save the data (using both the .npy for the points and the .yaml file for the characteristics of the database).
